Detailed Description

Generates images based on the Graph class.

Generates ".dot" markup files based on the directed graph and saves these files to in the "data/dot/" folder, then generates an image based on the file and stores the image in the "images/" folder.

def graphviz.Graphviz.add_path (   self,
  nodes,
  color_node,
  color_edge 
)

Add a path to be painted with the specified colors.

Note that _container_paths is FIFO, so the first paths to be place in will be painted first. Therefore, items at the end of the container will be painted on top of item in the front of the container.

Precondition:
The nodes and edges between them exist in the graph data.
Postcondition:
The specified path and colors will be added to _container_paths.
Parameters:
selfThe object pointer.
nodesString of nodes in the path, i.e. "ABC" for the path A to B to C.
color_nodeThe color that the nodes of the path will be painted.
color_edgeThe color that the edges of the path will be painted.

Generates an image for the graph data with the supplied name.

Creates the dot markup file for the graph in _directoryData and the image in _directory_images. Note that if the frames flag is set, the generated image is to be part of a set, so the name will be "name-(frame #)".

Precondition:
The graph data exist.
Postcondition:
An image of the graph will be located in _directory_images.
Parameters:
selfThe object pointer.
nameThe name of the image file.
frameDenotes whether the image is to be part of a set.
